Key Features:

  • User-Friendly Interface: Created an intuitive layout that enhances user experience, making it easy for individuals to find relevant information quickly.
  • Emergency Instructions: Developed step-by-step guides for various emergency situations, such as choking, bleeding, and CPR, complete with images and video tutorials to improve understanding.
